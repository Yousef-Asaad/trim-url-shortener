# AI Usage

**Tool used:** Claude

**Where AI helped:**
- Generating boilerplate for the Express + TypeScript backend setup (config, server bootstrap, error middleware)
- CRUD generation for the `Link` and `Click` Mongoose models and the link controller/routes
- Generating the React + TypeScript frontend scaffolding (pages, components, API client)

**Where AI was corrected:**
- URL validation: the initial draft only checked for `http://`/`https://` prefixes via string matching; this was tightened to also reject malformed URLs (using the `URL` constructor) and explicitly exclude dangerous schemes like `javascript:`, `data:`, `file:`, and `vbscript:`.

**Not delegated:**
- Architecture decisions (folder structure, collection design, indexing strategy, redirect status code choice) were specified upfront and followed as given, not generated by AI.
